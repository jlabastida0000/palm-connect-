# Quick Start Guide

Build your integration with Palm's Partner API. This guide walks you through setting up and implementing your first integration.

## Prerequisites

Before you begin, ensure you have:

- A Palm Partner account
- Basic understanding of REST APIs
- Your preferred programming language ready

<Note>
Don't have an account yet? [Sign up for free](https://dashboard.palm.io/signup)
</Note>

## Getting Started

<Steps>
  ### Get your API keys
  
  1. Log in to the [Palm Dashboard](https://dashboard.palm.io)
  2. Navigate to Settings â†’ API Keys
  3. Generate keys for both test and production:
  ```bash
  # Test Environment
  pk_test_xxxxxxxxxxxxxxxxxxxx
  
  # Production Environment
  pk_live_xxxxxxxxxxxxxxxxxxxx
  ```

  ### Install the SDK (Optional)
  
  Choose your preferred language:
  
  ```bash
  # Node.js
  npm install @palm/node

  # Python
  pip install palm-api
  ```

  ### Make your first API call
  
  Test the API with a simple profile lookup:
  
  ```bash
  curl -X GET "https://api.getpalm.com/v1/profiles" \
    -H "Authorization: Bearer YOUR_API_KEY"
  ```

  ### Set up webhooks
  
  Configure your webhook endpoint to receive real-time updates:
  
  ```bash
  curl -X POST "https://api.getpalm.com/v1/webhooks" \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -d '{
      "url": "https://your-domain.com/webhooks/palm",
      "events": ["profile.updated", "compliance.updated"]
    }'
  ```
</Steps>
