# Palm Partner API

Welcome to the Palm API documentation. This API enables you to manage business profiles, track compliance, handle stakeholder information, and more.

## Base URL

```bash
https://api.palm.io/v1
```

## Integration Options

### REST API
```bash
# Using API Key authentication
curl -X GET "https://api.palm.com/v1/profiles" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### SDKs
```bash
# Node.js
npm install @palm/node

# Python
pip install palm-api
```

### Webhooks
```json
{
  "type": "profile.updated",
  "data": {
    "profileId": "prof_123abc",
    "changes": ["compliance.status"]
  }
}
```
