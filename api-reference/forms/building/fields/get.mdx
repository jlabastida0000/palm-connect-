---
title: 'Get Form Field'
description: 'Retrieve a specific field from a form'
api: 'GET /v1/forms/{formId}/fields/{fieldId}'
---

Retrieve detailed information about a specific field in a form.

### Path Parameters

<ParamField path="formId" type="string" required>
  ID of the form
</ParamField>

<ParamField path="fieldId" type="string" required>
  ID of the field to retrieve
</ParamField>

### Request Examples

<CodeGroup>
```bash cURL
curl "https://api.palm.com/v1/forms/form_abc123/fields/field_xyz789" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript Node
import { Palm } from '@palm/node';

const palm = new Palm('YOUR_API_KEY');

const field = await palm.forms.fields.get('form_abc123', 'field_xyz789');
```
</CodeGroup>
