---
title: 'List Form Fields'
description: 'Get all fields in a form'
api: 'GET /v1/forms/{formId}/fields'
---

Retrieve a list of all fields in a form.

### Path Parameters

<ParamField path="formId" type="string" required>
  ID of the form to get fields from
</ParamField>

### Query Parameters

<ParamField query="type" type="string">
  Filter fields by type
</ParamField>

<ParamField query="page" type="number">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="number">
  Number of fields per page (default: 50, max: 100)
</ParamField>

### Request Examples

<CodeGroup>
```bash cURL
curl "https://api.palm.com/v1/forms/form_abc123/fields?type=select" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript Node
import { Palm } from '@palm/node';

const palm = new Palm('YOUR_API_KEY');

const fields = await palm.forms.fields.list('form_abc123', {
  type: 'select',
  page: 1,
  limit: 50
});
```
</CodeGroup>
