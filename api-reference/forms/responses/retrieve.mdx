---
title: 'Retrieve Responses'
description: 'Get form responses'
api: 'GET /v1/forms/{formId}/responses'
---

Retrieve responses for a form with filtering and pagination options.

### Path Parameters

<ParamField path="formId" type="string" required>
  ID of the form to get responses from
</ParamField>

### Query Parameters

<ParamField query="page" type="number">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="number">
  Number of responses per page
</ParamField>

<ParamField query="status" type="string">
  Filter by response status
</ParamField>

### Request Examples

<CodeGroup>
```bash cURL
curl "https://api.palm.com/v1/forms/form_abc123/responses?page=1&limit=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript Node
import { Palm } from '@palm/node';

const palm = new Palm('YOUR_API_KEY');

const responses = await palm.forms.responses.list('form_abc123', {
  page: 1,
  limit: 10,
  status: 'completed'
});
```
</CodeGroup>
