## Delete Transactions

`DELETE /transactions/{id}`

Deletes a single transaction based on the ID supplied.

### Path Parameters

- `id` (required): ID of the transaction to delete.

### Example Request

```plaintext
DELETE https://api.palmconnect.dev/api/accounting/v1/transactions/123
```

### Responses


- `204` (No Content): Transaction deleted successfully.
- `400` (Bad Request): Unexpected error.
```
{
  "error": 400,
  "message": "Unexpected error occurred while deleting the transaction."
}
