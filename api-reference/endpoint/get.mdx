## Get Transactions

`GET /transactions`

Returns a list of Transaction objects.

### Example Request

```plaintext
GET https://getpalm.com/transactions?limit=5

```
### Example Response

Responses
- `200` (OK): Transactions response.
- `400` (Bad Request): Unexpected error.

```json

{
  "error": 400,
  "message": "Unexpected error occurred while retrieving transactions."
}
