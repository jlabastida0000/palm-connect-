## Get Transactions

`GET /transactions`

Returns a list of Transaction objects.

### Query Parameters

- `limit` (optional): The maximum number of transactions to return (default: 10)
- `offset` (optional): The offset for pagination (default: 0)

### Example Request

```plaintext
GET https://api.palmconnect.dev/api/accounting/v1/transactions?limit=5&offset=10
```
### Example Response

[
