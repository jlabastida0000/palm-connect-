## Create Transactions

`POST /transactions`

Creates a new transaction(s) for your merchant.

### Request Body

The request body should contain a JSON object representing the transaction to be created.

#### Transaction Properties

- `name` (required): The name of the transaction.
- `tag` (optional): Tag to specify the type of transaction.

### Example Request

```plaintext
POST https://getpalm.com/transactions
Content-Type: application/json

{
  "name": "New Transaction",
  "tag": "expense"
}
```
### Example Response

```json
[
    {
        "id": "11",
        "amount": 200.75,
        "date": "2024-03-12T14:15:00Z",
        "description": "Expense reimbursement"

    {
        "id": "12",

        "description": "Gasoline purchase"
    },
    ...
