---
title: 'Retrieve Listing'
description: 'Get detailed information about a specific business listing'
api: 'GET /api/v1/listings/{listingId}'
---

Retrieve detailed information about a specific business listing using its unique identifier.

## Path Parameters

<ParamField path="listingId" type="string" required>
  Unique identifier of the business listing
</ParamField>

## Response

<ResponseField name="data" type="BusinessListing">
  Detailed business listing object
</ResponseField>

<RequestExample>
```bash Example Request
curl --request GET \
  --url 'https://api.getpalm.com/v1/listings/0d9f5013-2838-4e70-8375-e0ee3fef6bad' \
  --header 'Accept: application/json'
```
</RequestExample>
